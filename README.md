# Лабораторные работы по статистике и анализу данных на R

Репозиторий содержит лабораторные работы, реализованные на R и запускаемые в изолированных Docker-контейнерах.

## Состав

| Директория | Тема |
|------------|------|
| [`3/`](3/) | Построение правильно специфицированной регрессии (датасет `diamonds`) |
| [`5/`](5/) | Логистическая регрессия и ROC-анализ (датасет Titanic) |
| [`6/`](6/) | Кластерный анализ финансовых показателей компаний |

## Требования

- [Docker](https://docs.docker.com/get-docker/) + [Docker Compose](https://docs.docker.com/compose/)
- `make`

## Запуск

```bash
# Сборка общего базового образа (однократно)
make base

# Запуск конкретной лабораторной
make l3       # Лаб. 3
make l5       # Лаб. 5
make l6       # Лаб. 6

# Принудительная пересборка образа перед запуском
make l3-build
make l5-build
make l6-build
```

Результаты (графики) сохраняются в соответствующих директориях `<номер>/output/`.

## Инфраструктура

| Файл | Назначение |
|------|------------|
| [`Dockerfile.base`](Dockerfile.base) | Общий базовый образ (`r-base:4.5.3` + системные зависимости + R-пакеты) |
| [`Dockerfile`](Dockerfile) | Параметризованный образ для каждой лабораторной (`ARG SCRIPT`) |
| [`docker-compose.yml`](docker-compose.yml) | Сервисы `lab3`, `lab5`, `lab6` |
| [`Makefile`](Makefile) | Сокращённые команды сборки и запуска |
